# DocumentConverter 测试

这是一个 **Markdown 转换测试** 文档，用来验证 _Word_ 和 _PDF_ 转换功能。

## 一、功能介绍

支持以下 Markdown 语法：

1. 标题（H1-H6）
2. **粗体** 与 *斜体*
3. `行内代码` 和代码块
4. 表格、列表、引用
5. 链接和图片

## 二、代码示例

下面是一段 JavaScript 代码：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('DocumentConverter');
```

## 三、对比表格

| 格式 | 优点 | 缺点 |
|------|------|------|
| Word | 易于编辑 | 文件较大 |
| PDF  | 排版固定 | 不易修改 |
